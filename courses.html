Python基础知识

1.1 Python简介

Python是一种高级、解释型、面向对象的编程语言，具有简洁的语法和强大的功能。它广泛应用于数据分析、人工智能、Web开发等领域。

1.2 安装Python

推荐使用Miniconda来管理Python环境，这样可以避免版本冲突问题。

Miniconda环境搭建

2.1 什么是Miniconda

Miniconda是一个轻量级的Anaconda版本，只包含conda和Python。它允许用户创建和管理多个独立的Python环境，每个环境可以安装不同版本的包。

2.2 Windows系统安装

安装步骤：

  1. 访问Miniconda官方网站
  2. 下载Windows版本的安装包（推荐使用Python 3.9或更高版本）
  3. 运行安装程序，按照提示进行安装
  4. 勾选"Add Miniconda3 to my PATH environment variable"选项
  5. 完成安装后，打开命令提示符或PowerShell
  6. 输入conda --version验证安装成功

2.3 macOS系统安装

安装步骤：

  1. 访问Miniconda官方网站
  2. 下载macOS版本的安装包
  3. 打开终端，进入下载目录
  4. 运行安装脚本：bash Miniconda3-latest-MacOSX-x86_64.sh
  5. 按照提示进行安装
  6. 重启终端，输入conda --version验证安装成功

2.4 Linux系统安装

安装步骤：

  1. 访问Miniconda官方网站
  2. 下载Linux版本的安装包
  3. 打开终端，进入下载目录
  4. 运行安装脚本：bash Miniconda3-latest-Linux-x86_64.sh
  5. 按照提示进行安装
  6. 重启终端，输入conda --version验证安装成功

2.5 环境管理

# 创建新环境 conda create -n myenv python=3.9 # 激活环境 conda activate myenv # Windows source activate myenv # macOS/Linux # 安装包 conda install numpy pandas matplotlib # 列出已安装的包 conda list # 导出环境 conda env export > environment.yml # 删除环境 conda env remove -n myenv

数据科学常用库

3.1 NumPy

NumPy是Python中用于科学计算的核心库，提供了高性能的数组对象和数学函数。

import numpy as np # 创建数组 arr = np.array([1, 2, 3, 4, 5]) print(arr) # 数组运算 result = arr * 2 print(result) # 矩阵运算 matrix1 = np.array([[1, 2], [3, 4]]) matrix2 = np.array([[5, 6], [7, 8]]) product = np.dot(matrix1, matrix2) print(product)

3.2 Pandas

Pandas是用于数据处理和分析的强大库，提供了DataFrame数据结构。

import pandas as pd # 创建DataFrame data = { '姓名': ['张三', '李四', '王五'], '年龄': [25, 28, 30], '成绩': [85, 92, 78] } df = pd.DataFrame(data) print(df) # 数据查询 print(df[df['成绩'] > 80]) # 数据统计 print(df.describe())

3.3 Matplotlib

Matplotlib是Python中最常用的绘图库，可以创建各种类型的图表。

import matplotlib.pyplot as plt import numpy as np # 生成数据 x = np.linspace(0, 10, 100) y = np.sin(x) # 创建图表 plt.figure(figsize=(10, 6)) plt.plot(x, y, label='sin(x)', color='blue', linewidth=2) plt.xlabel('x') plt.ylabel('y') plt.title('正弦函数图像') plt.legend() plt.grid(True) plt.show()

3.4 安装常用库

# 安装单个库 conda install numpy # 安装多个库 conda install numpy pandas matplotlib scikit-learn # 使用pip安装 pip install requests
